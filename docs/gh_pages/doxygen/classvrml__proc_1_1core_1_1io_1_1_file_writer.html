Detailed Description

template<typename T>
class vrml_proc::core::io::FileWriter< T >

Interface for writing an object of type T to a file.

This abstract class defines a contract for file writers that serialize or export data to a file. Implementations must define the Write() method to handle specific formats or file types.

Template Parameters
TThe type of the object that will be written to the file.
