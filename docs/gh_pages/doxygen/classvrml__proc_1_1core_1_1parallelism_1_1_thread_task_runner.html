template<typename TaskType, typename ResultType>
void vrml_proc::core::parallelism::ThreadTaskRunner< TaskType, ResultType >::Run ( const std::vector< TaskType > & tasks,
std::vector< ResultType > & results )
inline

Runs a list of tasks in parallel using m_threads threads and stores their results in the results vector.

Each task from tasks[i] is executed in parallel and its result is stored at results[i].

Parameters
tasksvector of callable tasks, each must return a result compatible with ResultType
resultsoutput vector, where task results are stored; resized to match tasks.size()
Note
This function is thread-safe internally. Do not access results concurrently from other threads during execution.
