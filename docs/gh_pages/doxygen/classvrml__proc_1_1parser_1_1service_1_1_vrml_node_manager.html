Detailed Description

Manages DEF-defined VRML nodes, enabling lookup and registration.

This class stores references to VRML nodes defined using the DEF keyword, allowing later retrieval through USE references. It is used to maintain a mapping between node identifiers and their corresponding nodes without copying them.
